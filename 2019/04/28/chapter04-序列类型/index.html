4-1 python中的序列分类

纬度一:根据是否可以存放任意数据类型来划分

  • 容器序列:可以存放任意数据类型
    • list tuple deque
  • 扁平序列:只能存放特定的数据类型
    • str bytes bytearray array.array

纬度2:根据存放的数据类型是否可以被修改

  • 可变序列:可以增删改
    • list deque bytearray array
  • 不可变序列:不可以增删改
    • str tuple bytes
